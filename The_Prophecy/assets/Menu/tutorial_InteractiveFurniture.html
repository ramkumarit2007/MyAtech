Introduction

This example shows how to incorporate different 3D models and movie texture for tracking. Using GPS tracking configuration, this tutorial negates the necessity of a tracking pattern. The user can take a picture and set it as the tracking pattern for the second stage â€“ dummy tracking. In this setting, the user can put 3D models onto the picture taken. 3 gestures have been included and enabled in this tutorial, namely the drag, pinch and rotation gesture. A screenshot can also be saved by pressing a button.
